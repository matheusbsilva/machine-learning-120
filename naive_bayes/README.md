# Naive Bayes

## Bayes' Theorem
Bayes’ Theorem finds the probability of an event occurring given the probability of another event that has already occurred. Bayes’ theorem is stated mathematically as the following equation:

```
P(A|B) = P(B|A) * P(A) / P(B)
```
